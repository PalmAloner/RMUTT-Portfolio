เว็บไซต์พอร์ตฟอลิโอส่วนตัว

เว็บไซต์นี้เป็นโปรเจกต์แรกและโปรเจกต์หลักของผม ที่สร้างขึ้นเพื่อแสดงข้อมูลส่วนตัว ทักษะ และผลงาน สำหรับการสมัครเข้าศึกษาต่อที่ มหาวิทยาลัยเทคโนโลยีราชมงคลธัญบุรี คณะวิศวกรรมศาสตร์ สาขาวิศวกรรมทั่วไป (วิศวกรรมคอมพิวเตอร์)

เทคโนโลยี

HTML, CSS, JavaScript

ระยะเวลา

สร้างในปี 2024-2025

บทบาท

ออกแบบและพัฒนาเอง

คุณสมบัติของเว็บไซต์

  • หน้าแสดงข้อมูลส่วนตัวและประวัติการศึกษา
  • หน้าแสดงทักษะและความสามารถต่างๆ
  • หน้าแสดงผลงานและเกียรติบัติ
  • หน้าติดต่อพร้อมข้อมูลการติดต่อ
  • ออกแบบให้ใช้งานได้บนมือถือและคอมพิวเตอร์
  • มี Animation และ Effects สวยงาม

สิ่งที่ได้เรียนรู้

  • การเขียน HTML เพื่อสร้างโครงสร้างเว็บไซต์
  • การใช้ CSS ในการตกแต่งและจัดวางหน้าเว็บ
  • การใช้ JavaScript เบื้องต้นเพื่อสร้าง Interaction
  • การออกแบบ Responsive Design
  • การแก้ปัญหาเมื่อเจอ Bug หรือข้อผิดพลาด
  • ความอดทนและความตั้งใจในการทำงาน
